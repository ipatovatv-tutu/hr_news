# Описание бота для Telegram (BotFather)

Скопируй нужный блок в BotFather: **@BotFather** → твой бот → **Edit Bot** → **Edit Description** или **Edit About**.

---

## Short description (до 120 символов)

Показывается в чате с ботом и в поиске.

```
HR-дайджест: новости по персоналу, карьере и рынку труда. Команда /digest — прислать дайджест сейчас.
```

Вариант короче (если не влезает):

```
Дайджест HR-новостей (vc.ru, Хабр Карьера). Напиши /digest — пришлю дайджест.
```

---

## Description (до 512 символов)

Показывается на странице бота (профиль).

```
HR-дайджест — подборка новостей для HR и руководителей.

Что внутри: тренды по персоналу, рынок труда, рекрутинг, управление людьми, ИИ в HR, коммуникации, L&D. Источники: vc.ru (Карьера), Хабр Карьера.

Как пользоваться:
• Напиши /digest или /дайджест — пришлю свежий дайджест в этот чат.
• По расписанию дайджест приходит сам: понедельник и четверг, 09:00 МСК.

Бот не собирает личные данные и не хранит переписку.
```

---

## Команды бота (BotFather → Edit Commands)

В BotFather: **Edit Bot** → **Edit Commands** — вставь:

```
digest - Прислать HR-дайджест сейчас
```

Так в Telegram под полем ввода появится кнопка с командой **/digest**.
