# hr_news
Hr_news — бот для HR, который собирает новости из RSS и открытых источников и превращает их в понятные сигналы: что происходит на рынке, где риски и какие тренды влияют на людей и процессы. Меньше инфошума — больше контекста для быстрых и осознанных решений.м и помочь быстрее принимать решения, опираясь на контекст, а не на догадки.
